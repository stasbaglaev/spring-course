#### Задание
* создать еще один класс (жанр музыки)
* изменить класс MusicPlayer так, чтобы он хранил в себе не отдельные поля для музыкальных жанров, а хранил в себе список жанров (list)
* создать конструктор класса MusicPlayer, чтобы он ожидал на вход список музыкальных жанров
* создать бины для классов жанров музыки и музыкального плейера
* создать бин представляющий из себя список музыкальных жанров и внедрить его в бин MusicPlayer
* создать метод playMusic() так, чтобы на вход он воспроизводил песню случайного жанра (исп класс Random)

----
Аннотации @Component, @Autowired, @ComponentScan не использовать! Создавать бины и внедрять зависимости вручную в конфигурационном файле!
