#### Задание
* создать список из 3 песен в каждом жанре музыки (по 3 песни на каждый жанр)
* в MusicPlayer должны внедряться бина ClassicalMusic и RockMusic
* создать перечисление с двумя жанрами музыки - CLASSICAL и RockMusic
* создать метод playMusic() так, чтобы на вход он принимал Enum с жанрами музыки
* взависимости от указанного значения перечисления (CLASSICAL или RockMusic), метод playMusic()должен воспроизвести случайную одну песню из указанного жанра (исп класс Random)
